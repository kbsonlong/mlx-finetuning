## 🚀 快速开始

### 第一步: 配置本地模型 ⚠️ 重要

**本项目使用本地已下载模型，避免长时间下载等待**

1. 阅读 [配置文件模板](./05-config-templates.md)
2. 创建 `config/model_config.yaml` 并填写模型路径
3. 运行 `python scripts/verify_model.py` 验证模型
4. 确认模型文件完整

### 第二步: 审核方案
1. 阅读 [集成方案](./01-integration-plan.md)
2. 确认技术路线可行
3. 评估资源和时间

### 第三步: 了解基准测试
1. 阅读 [基准测试方案](./04-benchmark-plan.md)
2. 理解需要收集的基线数据
3. 准备测试环境

### 第四步: 查看计划
1. 阅读 [执行计划](./02-execution-plan.md)
2. 确认任务分解合理
3. 检查依赖项准备

### 第五步: 了解技术
1. 阅读 [技术架构](./03-technical-architecture.md)
2. 理解系统设计
3. 准备开发环境

### 第六步: 了解实验协议
1. 阅读 [实验协议](./06-experiment-protocol.md)
2. 理解数据版本控制要求
3. 熟悉评估标准

### 第七步: 开始执行
审核通过后，按执行计划**阶段 0**开始实施
